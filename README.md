# ios7crypt-rs: IOS7Crypt in Rust

# EXAMPLES

```console
$ ios7crypt -e monkey
1308181c00091d

$ ios7crypt -d 060b002f474b10
monkey

$ ios7crypt -h
Usage: ios7crypt [options]

Options:
    -h, --help          print usage info
    -v, --version       print version info
    -e, --encrypt VAL   encrypt a password
    -d, --decrypt VAL   decrypt a hash
```
