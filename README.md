# ios7crypt-rs: IOS7Crypt in Rust

# Example

```console
target/debug/ios7crypt -e monkey
1308181c00091d

target/debug/ios7crypt -d 060b002f474b10
monkey
```

# Requirements

* [Rust](http://www.rust-lang.org/) 1.16.0+

# BUILD

```console
$ cargo build
```

# UNIT TEST

```console
$ cargo test
```
